# To-Do List App

A simple, responsive, and interactive To‑Do List web application built with **HTML, CSS, and JavaScript**.  
It allows users to add tasks, edit them, delete them, mark them as completed, and track completed/uncompleted task counters.

---

##  Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed / uncompleted  
- Live task counters  
- Responsive UI  
- Smooth UX animations  
- Clean code architecture  

---

##  Technologies Used

- **HTML5**  
- **CSS3**  
- **Vanilla JavaScript**

##  Installation & Usage

1. Clone the repository:

git@github.com:NellaDev/todo-list.git

2. Open the folder:

cd todo-list

3. Launch the app by opening **index.html** in a browser.
